# RedisWebApp

A robust C# .NET 10 Web API built using Clean Architecture principles to ensure maintainability, scalability, and loose coupling.

## 🚀 Key Features
* **Clean Architecture:** Divided into `Api`, `Application`, `Domain`, and `Infrastructure` layers.
* **Database & ORM:** PostgreSQL managed via Entity Framework Core migrations.
* **Caching Layer:** Integrated Redis caching for high-performance data retrieval.
* **API Documentation:** OpenAPI / Swagger UI support.
